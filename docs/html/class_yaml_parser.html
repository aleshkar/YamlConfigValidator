Detailed Description

Строит дерево из упрощенного YAML-файла.

Парсер специально сделан небольшим и предсказуемым. Он не пытается покрыть весь стандарт YAML, но подходит для конфигов приложений и DevOps-конфигов, которые используются в этом учебном проекте.
